## How to run

Please follow the below instructions to run this project in your machine:

1. Clone this repository
   ```sh
   git clone https://github.com/learnwithsumit/css-flexbox-tutorial.git
   ```
2. Watch the youtube tutorial on this topic - https://youtu.be/kRS5ficucNM.
3. Double click any of the html file to see output as described in youtube tutorial.
